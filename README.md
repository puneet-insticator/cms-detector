# About CMS Detector
CMS Detector is a chrome extension that detects the Content Management Systems behind Websites and hint it with an icon in the chrome addressbar.

# Which systems are supported?
Typo3<br>
WordPress<br>
Joomla<br>
Drupal<br>
Contao<br>
